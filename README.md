# ai-product-projects

设计背景出身的产品经理，正在系统性转型 AI 方向。通过独立构建多个 AI Agent 项目，实践从需求定义、Agent 架构设计、Prompt 工程到效果评估的全流程。

## 项目总览

### 🔍 项目A：AI竞品分析助手
**平台**：Dify 工作流  
**状态**：进行中

基于 Dify 构建的多模块竞品分析系统，输入产品名称，自动完成信息采集 → 用户评论聚类 → PRD 草稿生成的全流程闭环。

- **模块1**：竞品信息采集与多维分析（SWOT、功能对比、用户口碑）
- **模块2**：用户评论自动采集与痛点聚类（自建采集 API + LLM 聚类分析）
- **模块3**：PRD 草稿自动生成

核心亮点：坚持全自动化数据采集，拒绝手动灌数据方案；经历了从手动采集 → Brave 搜索 → 自建 API 的三轮迭代演进。

---

### 📰 项目B：AI新闻聚合与行业洞察
**状态**：规划中

自动追踪 AI 行业动态，提取关键信号，生成结构化洞察报告。

---

### 🎨 项目C：设计师AI工作流Agent
**状态**：规划中

面向设计师的 AI 提效工具，结合设计行业 know-how 与 Agent 能力，覆盖从需求 brief 到交付物整理的工作流。

---

### 📚 项目D：企业知识库智能问答Agent
**状态**：规划中

基于 RAG 的企业知识库问答系统，实践知识库构建、向量检索、Rerank 策略的全链路。

---

## 技术栈

- **AI 工作流平台**：Dify
- **Agent 框架**：OpenClaw（腾讯云自部署）
- **AI 工具**：Claude API / Claude Code
- **通信渠道**：飞书（Lark）
- **基础设施**：腾讯云轻量服务器
